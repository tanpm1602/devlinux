# Các bài tập về Bitwise, Array và Pointer

## Giới Thiệu
Dự án này chứa các bài tập và file code về Bitwise, Array và Pointer của khóa học lập trình C/C++ trên Devlinux

## Mục Đích
Để thực hành các kỹ năng lập trình và nắm vững các khái niệm cơ bản.

## Nội Dung (What)
- `bitwise.c` - Các bài tập được trình bài trực tiếp trong hàm main()
- `array.c` và `pointer.c` - Các bài tập được viết thành các hàm, và khi chạy chương trình sẽ cần phải truyền tham số vào hàm main

## Cách Sử Dụng
Cần build ra từng file thực thi từ các file .c sau đó chạy các file thực thi mong muốn (build bằng gcc).
